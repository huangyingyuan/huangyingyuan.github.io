“机器学习”之 从入门到放弃(1)

发表于2024-07-15|更新于2024-07-15
|浏览量:

1，“机器学习”的基础定义

从广义上来说：机器学习是让机器学习，实现直接用编程无法实现的功能的做法

从意义上来说：是利用数据，训练出模型，实现数据预测的效果

总的来说：机器学习是由“预测”与“训练”的过程，得出未知的定律

让我们将其与人学习进行对比：

2，Python机器学习的六大库

1、NumPy

如果没有 NumPy，数据科学将是不完整的。它是一个可以进行科学计算的 Python 软件包。NumPy 是一个神奇

的多维数组对象库。它们协同工作，降低了程序的计算复杂性。

2、SciPy

SciPy 是 Python 人工智能项目的另一个热门库，也是涉及数学和工程领域中， Python 程序员科学和重度计算的

首选。它提供了数值优化和集成的例程，对于初学者来说非常友好。

3、Scikit-Learn

这个库建立在 NumPy 和 SciPy 之上，主要用于监督和无监督学习。它是一个用于数据挖掘和数据分析的完美工

具。

4、Pandas

Pandas 是开源的 Python 软件包，使程序员能够对数据进行操作和分析。它具有高效的数据探索和可视化功能，

并提供高级数据结构和多种工具，可用于密切处理多个数据集。

我之前用来识别excel大型的数据，将数据分类获取，处理大量数据挺好用

5、Keras

Keras 是一个运行在 TensorFlow 上的 API。Keras的重点是让开发者快速实验人工智能。这个库的用户体验比

TensorFlow 好得多——因为它是用 Python 开发的，所以比其他工具更容易理解。

6、Matplotlib

所有库中最强大的是 Matplotlib。它提供了数据可视化和探索的功能，以及图表、直方图和散点图等，以定制

Python AI 项目。Matplotlib 有助于在更短的时间内快速操作数据进行可视化展示。

3，机器学习的相关科目

1，线性代数

这门科目我也不想说什么 What can i say? 只能说“不错”，我“爱”老师

说实在的，在基础的机器学习可能你感受不到线性代数的作用，但是在深入地去了解机器学习后，或者在计算机大

大小小方面来说，还是会经常交面的,从素未相识到老朋友

2，微积分

是计算机的求多种数学/物理模型算法求解的核心

3，典型算法

处理分类问题的常用算法包括：

逻辑回归(工业界最常用)，支持向量机，随机森林，朴素贝叶斯(NLP中常用)，深度神经网络(视频、图片、语音等

多媒体数据中使用)。

处理回归问题的常用算法包括：

线性回归，普通最小二乘回归（Ordinary Least SquaresRegression），逐步回归（Stepwise Regression），

多元自适应回归样条（MultivariateAdaptive Regression Splines）

处理聚类问题的常用算法包括：

K均值（K-means），基于密度聚类，LDA等等。

降维的常用算法包括：

主成分分析（PCA）,奇异值分解（SVD） 等。

推荐系统的常用算法：

协同过滤算法

模型融合(model ensemble)和提升(boosting)的算法包括：

bagging，adaboost，GBDT，GBRT

其他很重要的算法包括：EM算法等等。

4,机器学习的过程

明确目标：

就像你要教小孩子学骑自行车，首先得告诉他目标是学会骑车，而不是跑步或者游泳。在机器学习

中，你得清楚你想要机器帮你做什么，比如识别图片中的物体，或者预测未来的天气。

2.数据收集

收集资料：教小孩子骑车，你需要给他一辆自行车和一些安全装备。在机器学习中，你需要收集相关的数据，这些

数据就像是机器学习的“自行车”和“安全装备”，帮助它学习如何解决问题。

3.数据清洗

整理资料：小孩子学骑车时，你得确保自行车没有问题，比如轮胎气要足，刹车要灵。在机器学习中，你需要检查

数据，去掉那些不完整、错误或者重复的部分，确保数据的质量。

4.数据评估

检查资料：在小孩子开始骑车之前，你会检查一下他是否准备好了，比如头盔戴好了没有。在机器学习中，你需要

评估数据是否足够好，是否能够帮助机器学习到有用的信息。

5.扩大模型

丰富教材：教小孩子骑车，你可能还会给他看一些骑车的视频，或者让他练习平衡。在机器学习中，你可能需要对

数据进行一些加工，比如增加一些新的特征，或者调整数据的格式，让机器学习的内容更加丰富。

6.模型正则化与超参数

调整学习方法：教小孩子骑车，你得告诉他一些规则，比如不要骑得太快，注意安全。在机器学习中，你需要给模

型一些规则，防止它学得太死板（过拟合）。同时，你还需要调整一些学习参数，就像是调整学习方法，让机器学

习得更好。

总结

整个机器学习的过程，就像是教一个小孩子学习新技能。首先，你要告诉他学什么（定义问题），然后给他教材

（数据收集），帮他整理教材（数据清洗），检查教材是否合适（数据评估），丰富教材内容（扩大模型），最后

调整他的学习方法（模型正则化与超参数）。通过这些步骤，小孩子（机器）就能学会解决问题了。

这一期的机器学习总能让你有个大概的了解了吧，等我后续更新，继续进一步了解
